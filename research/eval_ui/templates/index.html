AI Agent Evaluation

Help us understand how people experience different AI conversation styles. Your responses are anonymous and used for research purposes only.

How it works

  1. Chat with Agent A for 5 turns
  2. Answer 5 quick questions about Agent A
  3. Chat with Agent B for 5 turns
  4. Answer the same 5 questions about Agent B
  5. Tell us which agent you preferred

Takes about 10-15 minutes. Just chat naturally — there are no wrong answers.
